## Recording Tips

1. **Before Recording:**
   - Clear browser cache and reload
   - Close unnecessary tabs
   - Hide bookmarks bar (Cmd+Shift+B on Mac)
   - Set browser zoom to 100%
   - Test audio levels

2. **During Recording:**
   - Speak clearly and at moderate pace
   - Pause briefly between sections
   - Use cursor to highlight key information
   - Don't rush - 2 minutes is enough

3. **Products to Demo:**
   - Choose a mid-priced product (₹500-₹1500)
   - Avoid products with extreme prices
   - Pick one with good competitor data

4. **What to Avoid:**
   - Don't mention the ₹89.99 issue if it's still there
   - Don't show console/developer tools
   - Don't mention "prototype" or "MVP"
   - Skip technical implementation details

## Alternative 90-Second Version (If Needed)

If you need a shorter demo, cut these sections:
- Detailed technology stack (just mention "Built on AWS")
- Competitor analysis details (just show it briefly)
- Focus on: Problem → Solution → AI Recommendation
